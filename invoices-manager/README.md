## Getting Started

- Clone repo

```
git clone git@gitlab.asoft-python.com:nhung.duong/react-training.git
```

- Checkout branch

```
git checkout feature/invoices-manager
cd invoices-manager
```

- Install dependencies

```
yarn install
```

- Run Webpage

```
yarn dev
```

Open http://localhost:3000/ in browser
